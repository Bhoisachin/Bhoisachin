## âœ¨ Key Features

This portfolio is a fully responsive static website with some dynamic features:

* **Responsive Design:** Looks great on all devices, from mobile phones to desktops.
* **Dark/Light Theme:** A theme toggler for comfortable viewing in any lighting.
* **Smooth Animations:** A smooth, sliding mobile menu for a better user experience.
* **Interactive Elements:** Hover effects and a custom modal for the contact form.
* **Dynamic Contact Form:** A contact form that sends messages directly to a Google Sheet using Google Apps Script.
